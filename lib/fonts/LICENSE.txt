Provenance: obtained from the Dear ImGui distribution (misc/fonts/Roboto-Medium.ttf).
Used by the MGB64 app shell as the embedded UI font. The .ttf byte source is NOT
tracked in this repository (which keeps no binary blobs); the tracked build input
is src/app/app_font.h, a compressed-base85 re-encoding generated by lib/imgui's
binary_to_compressed_c tool. Fetch Roboto-Medium.ttf upstream to regenerate.
No ROM-derived data is involved.
